# 优酷视频MP4文件修复工具

**场景**：春节期间帮老人家从优酷上缓存了一些戏曲视频，结果后来从手机导出之后使用其它播放器无法播放。作为技术行业的从业者怎能忍？:D 经过研究写了这么个工具，请自行编译。

**探索**：优酷对MP4源文件进行了简单的加密处理（其实谈不上加密，他们只是对MP4文件格式做了点小手脚），导致只能在优酷播放器里播放；经过工具修复后可在任意播放器里播放。
